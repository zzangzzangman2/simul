## 2026-07-21 — 장소형 게임 장면과 행동 시간 비용

- 중요한 기능을 일반 앱 화면·바텀시트로 띄우지 않고 게임 세계의 장소로 배치했다.
- 주식 목록과 종목 상세는 기존 소년 방 배경 위의 `내 방 · CRT 투자 단말` 프레임 안에서 열린다. 모니터 베젤, 장소 설명, 현재 시간이 유지된다.
- 신규 안건은 `우리 집 거실 · 가족회의` 장면으로 전환되며, 신문은 `우리 집 거실 · 저녁 신문` 장면의 식탁 위 지면으로 열린다.
- 조직 화면은 거실 사람들 장면, 일거리 화면은 집과 동네 장소 시계를 표시한다.
- 단순 열람·장소 이동은 시간을 소모하지 않는다. 결과가 생기는 행동만 안건 결정 30분, 가족 도움 30분, 일거리 완료 60분을 소모한다. 20:00을 넘으면 장 종료 시각으로 고정한다.
- 시장은 기존 3분 자동 틱과 `+1시간` 진행을 그대로 사용하고, CRT를 나갈 때 현재 시각을 저장한다.
- 360px에서는 장소명과 시계를 유지하고 부가 비용 배지만 숨겨 가로 넘침을 방지했다.
- 검증: Flutter 정적 분석 0건, 전체 테스트 47개 통과. 안건 30분·일거리 60분 실제 화면 반영과 전체 거실 신문 장면 360px 렌더링을 포함한다.
