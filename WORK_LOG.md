# 작업 이력

이 문서는 완료한 작업을 버전별로 남기는 변경 기록입니다. 현재 상태와 다음 작업은 `HANDOFF.md`, 장기 방향은 `PRODUCT_VISION.md`에서 관리합니다.

## 기록 규칙

- 사용자에게 보이는 기능이나 데이터가 바뀌면 항목을 추가합니다.
- 각 버전에는 날짜, 완료 내용, 검증 결과, 남은 일을 적습니다.
- 아직 완료하지 않은 내용은 `다음 버전 후보`에만 기록합니다.
- 커밋 해시나 PR이 있으면 함께 남깁니다.
- 패키지 버전과 배포 버전이 달라지면 둘 다 명시합니다.
