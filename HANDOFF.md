# 부자되기 시뮬레이션 인수인계

마지막 갱신: 2026-07-24

## 한 줄 상태

주 구현은 `flutter_app/`이다. 거래 가능한 대상은 가상기업뿐이며, 게임은 고정 출발 기업 30개에서 시작해 월드시드에 따라 신규상장·분사·유상증자·상장폐지가 이어지는 2000~2010 가상시장을 만든다.

저장소 루트의 기본 경로는 `/play/index.html`로 리다이렉트한다. 예전 React 게임과 실제 종가 원장, 실경영진·역사 이벤트 코드는 제거했다.

신규 게임은 TV 드라마 속 투자자를 보고 주식에 관심을 가진 주인공의 가족 대화와 투자학원 수업으로 시작한다. 외할아버지가 준 세뱃돈 10,000원은 어머니 명의 교육용 증권계좌 예수금과 `family_gift` 원장에 함께 기록한다. 아빠가 먼저 낸 학원비 1,000,000원은 별도 가족 채무이며, 거실 조직 화면에서 아빠를 선택한 뒤 회사 통장 잔액으로 전액 상환하면 `family_debt_repayment` 원장이 남는다. 증권 예수금은 자동 출금하지 않고 기존 저장에는 선물이나 채무를 소급 적용하지 않는다.

학원 수업은 한서윤 선생님과 주인공의 질문·답변이 번갈아 나오는 미연시 장면이다. 신규 플레이는 학원 마지막 이름 입력 직후 집으로 돌아가지 않고 시장 실습으로 바로 이어진다. 선생님은 오른쪽 아래의 큰 상체 캐릭터로 전환되고, 노란 강조 영역만 누를 수 있는 실습이 `주식 탭 → 한빛통신 → 현재가 → 차트·재무 → 사기 → 주문표` 순서로 진행된다. 마지막 주문표 버튼은 저장을 기다리지 않고 즉시 안내막을 닫으며 완료 상태는 비동기로 저장한다. 중간에 앱을 닫으면 다음 접속 때 처음부터 다시 시작하고, 기존 저장은 완료로 간주해 강제로 띄우지 않는다.

## 확정된 하루 흐름

1. 오늘 20:00 상태와 종가, 이미 공개된 사건을 저장한다.
2. 다음 날 08:00 준비 화면에 `조간신문과 오늘의 시장을 준비 중입니다…`를 표시한다.
3. Gemini는 전날 시장에서 공개된 사실만 요약한다.
4. 엔진은 오늘의 숨은 시나리오를 월드시드로 한 번 생성해 저장한다.
5. 유저는 전날 신문을 읽고, 원하면 오늘의 조사보고서를 1,200원에 구매한다.
6. 보고서는 최대 3개의 현장 징후만 주며 상승·하락, 성공 여부, 영향률, 미래 종가를 숨긴다.
7. 시간이 각 사건의 `revealMinute`를 지나면 속보 시트로 공개한다.

신문을 닫기 전부터 저장 상태는 다음 날 08:00이다. 신문 닫기 방식과 관계없이 날짜가 다시 진행되면 안 된다.

## 가상시장 구조

- `fixedFictionalCompanies`: 모든 게임에 고정으로 존재하는 30개 회사
- 시작 대표 회사: 한빛통신
- 시장: 미래시장, 도전시장
- 월드시드 변화: 일별 가격, 장중 경로, 사건 시기·성패, 분사 후보, 신규상장, 상장폐지
- 기업 계보: `generation`, `parentAssetId`, `listedOn`, `delistedOn`
- 기업 분석: 분기 재무, 컨센서스, 영업현금흐름, 수주잔고, 공급사·고객사·경쟁사·모자회사 관계
- 한 월드의 전체 종목 수: 고정 30개보다 크며 보통 70개 이상
- 연속 사건 수: 2000~2010 기간 3,000개 이상
- 기업행동: 액면분할, 배당, 유상증자, 물적분할, 인적분할, 신규상장, 상장폐지
- 주문장: 시장가·지정가, 가격·시간 우선 부분체결, 예수금·보유수량 예약, 취소, 15:00 자동 만료

업종 허용표가 사건 종류를 제한한다. 바이오 임상은 바이오·제약·의료에만, 선박 수주·후판·인도 문법은 조선·기계에만 붙는다. 공통 자금·규제·회계 문법도 회사 제품과 현재 상태를 반영해야 한다.

## 저장과 마이그레이션

현재 상태 스키마는 `v15`다.

- 새 게임은 매번 고유 `simulationSeed`를 만든다.
- 테스트·재현이 필요할 때만 `worldSeed`를 명시한다.
- `hiddenMarketScenario`는 같은 날짜에 다시 뽑지 않는다.
- 시나리오 보관함은 최근 32개, 뉴스 보관함은 최근 90개를 유지한다.
- 과거 실기업 포지션은 마이그레이션 때 원가를 현금으로 환급하고 제거한다.
- 과거 제어 회사는 한빛통신 가상세계 상태로 변환한다.
- 가상기업 포지션과 소수점 레거시 수량은 보존한다.
- 시장 실습은 신규 플레이만 미완료로 시작하며, 기존·마이그레이션 저장은 완료 상태를 유지한다.

## Gemini 경계

Gemini는 세계 결과를 결정하지 않는다. 엔진이 먼저 시나리오·가격·사건을 고정하고 Gemini는 전날 공개 사실을 기사 문장으로 정리한다. 호출 실패 시 결정론적 로컬 신문을 사용한다.

동적 기사 입력에는 오늘의 숨은 시나리오, 보고서 힌트, 영향률, 미래 결과를 넘기지 않는다.
