## D-014 — 국내 시장을 기본 무대로 하고 해외 종목은 별도 탭에 보존

- 상태: 채택
- 결정: 주가창 기본 화면은 국내 22개 개발용 팩으로 시작하고 KOSPI·KOSDAQ 필터를 제공하며, 기존 미국·일본 6개 종목은 `해외` 탭에 보존합니다.
- 이유: 국내 기업을 골라 성장·인수하는 핵심 재미에 집중하면서 이미 만든 해외 데이터와 향후 해외 진출 확장성을 잃지 않기 위해서입니다.
- 가격 규칙: 개입 전 거래일 마감값은 실제 일별 종가를 사용합니다. 시장 화면이 열려 있을 때만 현실 1초당 게임 1분이 진행하며 장중 가격은 실제 종가로 수렴합니다.
- 틱 움직임: 종목·날짜별 시드의 난수 충격, 단기 모멘텀, 변동성 군집과 드문 급변을 합성한 뒤 브리지 보정으로 실제 종가에 도달시킵니다. 반복 파동은 사용하지 않으며 실제 일봉과 게임용 1분봉을 분리합니다.
- 분봉 차트: 기본과 최소 단위는 1분이며, 실제 체결 분봉이 아닌 결정론적 게임 연출임을 화면에 구분합니다.
- 증권 UX: 토스증권처럼 가격→전일 대비→차트→기간→핵심 지표→사기/팔기 순서로 읽히게 하고, 주문은 10살 주인공 설정에 맞춰 부모 승인 시트로 연결합니다.
- 전 종목 확장: `scripts/fetch-krx-full-market.py`가 모든 평일 상장목록 합집합과 원시 종가를 수집합니다. KRX 계정과 재배포 권한 확인 전에는 생성물을 공개 저장소에 넣지 않습니다.
- 대체된 결정: D-004의 정규화 수정주가만 표시하는 방식은 보조 `adjustedIndex`로 남기고, 화면 가격은 원시 일별 종가로 전환합니다.

## D-015 — Flutter에서 실제 거래와 기존 웹 진행을 계승

- 상태: 대체
- 결정: 기본 사용자 경로인 Flutter에 실제 일별 종가를 앵커로 삼는 현재 장중 게임 체결가 기반 매수·매도, 거래비용, 포지션과 원화 AUM 원장을 연결하고 상태 스키마에 포함합니다. 기존 React 웹 v3 저장은 명시적인 가져오기 마이그레이션으로 계승합니다.
- 이유: 가족 프롤로그 뒤의 조사와 부모 승인 흐름이 실제 투자 결과로 이어지게 하고, 기술 전환 때문에 기존 플레이 진행을 잃지 않기 위해서입니다.
- 당시 결과: React 앱은 회귀 비교용으로 남고 기본 플레이와 저장의 기준을 Flutter로 옮겼습니다. 같은 저장을 중복 변환하거나 이전 진행을 조용히 삭제하면 안 됩니다. 실제 환율 원장이 없는 USD·JPY 주문은 참고 전용으로 잠가 외화 가격을 원화처럼 계산하지 않습니다.
- 대체된 결정: D-016의 Flutter v13 보존 마이그레이션이 현재 저장 기준을 계승합니다.

- 이유: 온보딩 대사, 실제 주문 가능 자금, 계좌 의미를 하나로 연결하면서 세뱃돈과 노동소득을 구분하기 위해서다.
- 결과: 신규 게임의 총현금과 증권 예수금은 각각 10,000원이고, 시작 장부에는 `외할아버지 세뱃돈 · 첫 투자금`이 남는다. 명시적으로 0원이나 테스트 자금을 넣는 생성 경로와 레거시 마이그레이션은 기존 잔액을 보존한다.

## D-026 — 투자학원 프롤로그와 아빠 선납 교육비

- 상태: 채택
- 날짜: 2026-07-23
- 결정: 프롤로그는 TV 드라마 속 투자자를 본 주인공의 투자 선언으로 시작하고, 가족의 권유에 따라 중·고등학생이 함께 듣는 투자학원 입문반에서 회사의 의미, 매수·매도, 시장가·지정가와 투자 이유 기록을 배운다. 아빠가 학원비 1,000,000원을 먼저 내고 신규 저장에는 같은 금액의 가족 채무를 기록한다.
- 상환 규칙: 거실 조직 화면에서 아빠를 선택해야 채무 카드가 보인다. 회사 통장 잔액이 채무 전액 이상일 때만 한 번에 갚으며 증권 예수금을 자동 출금하지 않는다. 성공 시 `family_debt_repayment` 원장과 관계 보상을 기록한다.
- 이유: “왜 아이가 바로 주식을 아는가”를 플레이 가능한 수업으로 설명하고, 아빠의 지원을 공짜 시작 자본이나 투자용 레버리지로 오해하지 않게 하기 위해서다.
- 호환성: 새 온보딩을 완료한 신규 저장에만 채무를 만들고, 플래그가 없는 기존 저장과 마이그레이션 기본값은 0원으로 유지한다.

## D-027 — 첫 주식시장 강제 클릭 실습

- 상태: 채택
- 날짜: 2026-07-24
- 결정: 투자학원 수업은 한서윤 선생님과 주인공이 질문·답변을 번갈아 하는 미연시 장면으로 구성한다. 신규 플레이는 학원 마지막 이름 입력 직후 집을 거치지 않고 시장 실습으로 바로 이동하며, 선생님은 오른쪽 아래의 큰 상체 캐릭터와 말풍선으로 전환해 `주식 탭 → 한빛통신 → 현재가 → 차트·재무 → 사기 → 주문표`를 차례로 가르친다.
- 상호작용 규칙: 안내 중에는 전체 화면과 뒤로가기를 막고 현재 단계의 노란 강조 영역만 누를 수 있다. 주문표의 마지막 설명까지는 실제 주문이 실행되지 않으며, 설명을 끝낸 뒤 플레이어가 수량·시장가·지정가·예상 금액·수수료를 직접 확인한다. 학원 대화 장면의 선생님은 주인공과 비슷한 크기로, 시장 안내의 선생님은 전신 대신 머리부터 허리 중심의 상체 크롭으로 표시한다.
- 저장 규칙: 마지막 주문표 설명 버튼은 저장 지연과 무관하게 안내막을 즉시 닫고 `marketTutorialSeen`과 완료 일자를 비동기로 저장한다. 중간 이탈이나 저장 실패는 완료로 보지 않아 다음 접속 때 처음부터 다시 시작한다.
- 호환성: `marketTutorialEligible`은 실제 신규 플레이에만 켠다. 기존 저장과 마이그레이션 기본 상태는 완료로 처리해 과거 플레이어를 강제로 잠그지 않는다.
