# 여자1 캐릭터 사용 메모

이 폴더는 현재 확정 전인 여자친구 NPC `여자1`의 원본 보관 폴더다.
지금 게임 화면에는 연결하지 않고, 이후 사용자가 등장 장면과 역할을 지정하면 그때 적용한다.

## 고정 디자인

- 성인 여성 캐릭터
- 긴 애쉬브라운 머리와 옆 포니테일
- 금색 마름모 머리핀 2개
- 올리브 골드색 눈
- 가슴이 작은 슬림 체형
- 얼굴, 체형, 머리와 눈 색은 의상별로 바꾸지 않는다.

## 파일

- `01_교복_표정.png`: 교복풍 의상
- `02_평상복_표정.png`: 외출용 평상복
- `03_홈웨어_표정.png`: 나시티와 돌핀팬츠 홈웨어

각 파일은 왼쪽 위부터 `기본`, `미소`, `활짝 웃음`, `울음`, `삐짐`, `수줍음` 순서의 3×2 표정 시트다.

## 나중에 적용할 때

1. 사용자가 여자1의 역할과 등장 장소를 확정한다.
2. 필요한 의상과 표정을 시트에서 개별 에셋으로 분리한다.
3. 대사창과 이벤트 장면에 연결한다.
4. 다른 여자 캐릭터를 만들 때 여자1의 얼굴이나 고유 머리 모양을 복제하지 않는다.
