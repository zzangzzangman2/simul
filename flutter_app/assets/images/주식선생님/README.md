# 주식 학원 선생님 캐릭터 메모

> 이 캐릭터는 **주식시장 입문 튜토리얼 전용 선생님**으로 사용한다.
> 신규 게임에서 주식 화면을 처음 열었을 때 매수·매도·차트·주문 방법을 설명하는 안내 캐릭터로 배치한다.

## 고정 설정

- 나이: 21세 성인 여성
- 역할: 주식 투자 학원 선생님 및 주식시장 튜토리얼 안내자
- 체형: 전체적으로 가늘고 긴 슬랜더 체형
- 얼굴: 보라색 눈, 차분하고 친근한 인상
- 머리: 짙은 자주빛 갈색의 긴 로우 포니테일
- 의상: 아이보리색 긴팔 랩 원피스, 차콜색 얇은 벨트, 검은색 낮은 굽 구두
- 파일 배경: 알파 투명

## 최종 파일

- `05_6자세_슬랜더_투명_최종.png`
- Flutter 에셋 경로: `assets/images/주식선생님/05_6자세_슬랜더_투명_최종.png`

포즈 순서는 왼쪽 위부터 다음과 같다.

1. 지시봉을 든 기본 자세
2. 옆의 차트를 가리키는 자세
3. 마커와 펼친 손으로 설명하는 자세
4. 태블릿을 든 자세
5. 팔짱을 낀 자신감 있는 자세
6. 한 손을 허리에 두고 핵심 내용을 강조하는 자세

## 나중에 적용할 때

1. 필요한 자세를 개별 PNG로 분리한다.
2. 주식시장 첫 진입 튜토리얼 화면에 우선 배치한다.
3. 표정이 추가로 필요하면 동일 얼굴과 체형을 고정한 채 별도 제작한다.
4. 현재 최종 캐릭터보다 체형을 굵게 바꾸지 않는다.
