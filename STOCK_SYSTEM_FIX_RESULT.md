| API | Origin·입력 크기·IP 요청 제한·구조화 출력 검증. |

## 리소스 사용

- 시장 자동 갱신은 1초마다 한 번만 발생합니다.
- 시장 화면 이탈, 주문 처리, 저장 처리 중에는 타이머를 중단합니다.
- 28개 종목의 하루 결정론적 분봉 경로는 현재 모바일 메모리 범위에서 작고 네트워크 요청이 없습니다.
- 동적 뉴스는 주말·일반 휴장일마다 호출하지 않으며, 실패해도 로컬 기사로 진행합니다.

## 회귀 검증

`flutter analyze`와 Flutter 전체 테스트가 통과했습니다. 테스트는 시장 화면에서 현실 1초 후 게임 1분 진행, 화면 이탈 중 무진행, 거래 검증, 월 결산, 채용·펀드, 저장 복구, 일요일 시작·월요일 08:00 전환, 캠페인 상한과 모바일 레이아웃을 포함합니다.

## 현재 모바일 LAN 확인

- 같은 Wi-Fi 휴대폰: `http://10.10.1.159:8000/play/index.html`
- 루트 `/`는 307로 `/play/index.html`에 이동하고, 플레이 화면은 200을 반환합니다.
- 제공 중인 `main.dart.js`는 방금 빌드한 파일과 SHA-256이 일치합니다.
