# 밸런스 기록

이 문서는 게임용 수치의 현재 값과 조정 이유를 기록합니다. 실제 회사의 내부 재무나 실제 주가로 해석하면 안 됩니다.

### 제품 제안

| 선택 | 비용 | 주요 효과 |

### 개발 문제와 출시

- 품질 보강: 80,000원, 품질 +16, 사기 +3, 위험 -5, 출시까지 4일
- 기능 축소: 20,000원, 진행 +24, 품질 -8, 시장성 -5, 출시까지 2일
- 개발 연장: 35,000원, 품질 +8, 위험 -3, 사기 -1, 출시까지 6일
- 출시 연기: 40,000원, 품질 +12, 위험 -7, 시장성 -4, 사기 -2
- 즉시 출시: 선점은 유지하지만 위험 +6

### 결과 계산

현재 초기 결과 점수는 기술 + 브랜드 + 사기 + 품질 + 시장성 - 위험 + 결정론적 변동값으로 계산합니다. 변동값은 `simulationSeed`와 프로젝트 경로에서 정해지며 범위는 -15~+15입니다.

결과 단계는 큰 성공, 안정적 성공, 엇갈린 출시, 실패 네 구간입니다. 각 구간은 현금, 월 매출, 브랜드, 사기와 시뮬레이션 기준지수를 동시에 바꿉니다. 임계값은 플레이테스트 전 임시값이며 공개 실제 확률이 아닙니다.

부동산은 취득가를 비시장 장부가로 기록하며 주식 운용 AUM에는 합산하지 않습니다. 예상 매각가는 취득 직후 90%에서 시작해 보유 1년마다 2%p 올라가고 115%에서 상한을 둡니다. 취득 후 30일 동안은 매각할 수 없습니다. 모든 값은 게임 밸런스용 가상 수치입니다.

### 성인 확률 오락

- 해금: 2010년이면서 플레이어 나이 20세 이상
- 빈도: 게임 월 1회
- 참가금: 최소 10,000원, `현금의 1%`와 `100,000원` 중 작은 값을 상한으로 적용
- 결과: 60%는 0배, 30%는 1.5배, 10%는 3배
- 평균 지급률: 75%, 평균 손실률: 25%
- 실제 결제, 광고 시청 보상, 실제 현금 환전은 없음

결과는 회사 시드·월·참가금으로 결정론적으로 계산해 같은 저장 입력에서 바뀌지 않습니다. 확률과 평균 지급률은 참여 확인 전에 항상 표시합니다.

## 조정 원칙

- 어떤 한 선택도 항상 최고 결과를 만들지 않게 시드와 누적 상태를 함께 봅니다.
- 비용만 다르고 결과가 같은 선택지는 만들지 않습니다.
- 실패해도 즉시 게임오버보다 회생 경로를 남깁니다.
- 가상 시장 데이터를 연결한 뒤 기준선 지수와 가상 기업경영 가격 변동 폭을 다시 보정합니다.
- 수치를 바꾸면 관련 엔진 테스트의 기대 의도와 이 문서를 함께 갱신합니다.
